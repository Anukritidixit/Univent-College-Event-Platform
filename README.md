# Univent - Multi-Institution Event Orchestration Platform

**Univent** is a B2B SaaS platform designed to digitize and centralize event management for educational institutions. It solves the fragmentation of college fests by providing a unified infrastructure for Organizers to host events and Students to participate effortlessly.

## Core Features
* **Multi-Tenancy Architecture:** capable of hosting multiple colleges on a single platform instance.
* **Role-Based Access Control (RBAC):** Secure hierarchical login system (Admin vs. Student) with protected routes.
* **Event Lifecycle Management:** Admin dashboard for creating, editing, and monitoring fest activities (In Progress).
* **Secure Authentication:** Implementation of JWT (JSON Web Tokens) and Bcrypt for data security.

## Tech Stack
* **Frontend:** React.js, Tailwind CSS, Axios
* **Backend:** Node.js, Express.js, REST API
* **Database:** MongoDB (Atlas)
* **Version Control:** Git & GitHub

## Future Roadmap
* Integration of Payment Gateways (Razorpay) for paid registrations.
* Real-time analytics dashboard for College Admins.
* Automated certificate generation for participants.
