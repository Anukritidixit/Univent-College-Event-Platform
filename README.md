#  Univent - College Event Management Platform

**Univent** is a B2B SaaS platform designed to streamline event management for colleges. It creates a seamless bridge between College Admins (organizers) and Students (participants).

##  Key Features (Implemented)
* **Professional MERN Architecture:** Separated Frontend/Backend structure.
* **Secure Authentication:** JWT-based login with Password Hashing (Bcrypt).
* **Role-Based Access Control (RBAC):** Distinct dashboards for **Admins** and **Students**.
* **Scalable Database:** MongoDB schema designed for high-volume event registration.

##  Upcoming Features (In Progress)
* **Event Creation Wizard:** Drag-and-drop interface for organizers.
* **Registration Gateway:** Instant student enrollment logic.
* **Data Analytics:** Admin dashboard for participation metrics.
